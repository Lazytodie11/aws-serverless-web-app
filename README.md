# AWS Serverless Web App

This project is an AWS-based serverless web application that I am building by integrating four coursework assignments into one portfolio-ready project.

## Project Goal

The goal is to turn separate AWS homework assignments into one unified project that can later be presented in:

- GitHub
- Resume
- Portfolio / interview discussion

## Planned Features

This project will include two main modules:

### 1. Num2Words API + DynamoDB History
- Accept an integer input
- Convert the number into English words
- Store input/output history in DynamoDB
- Provide a frontend page for user interaction

### 2. Movie Search API + OpenSearch
- Accept a keyword query
- Search movie data using OpenSearch
- Return matching movie results
- Provide a frontend page for search interaction

## Tech Stack

- Python
